```js
export function formatDate({
  date,
  options = {
    year: "numeric",
    month: "short",
    day: "numeric",
  },
  locale = "es-ES",
}) {
  return new Date(date).toLocaleDateString(locale, options);
}
```

## Uso

El método `toLocaleDateString` nos devuelve el objeto de `Date` como una cadena utilizando las convenciones locales, también podemos agregar argumentos los cuales nos permiten personalizar el comportamiento de la función.

> Ingresar fecha en una cadena de texto con el siguiente formato "Año/Mes/Día".

```js
console.log(formatDate({ date: "2025/04/04" }));
// 4 abr 2025
```
