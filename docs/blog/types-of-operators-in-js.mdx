## Operadores Aritméticos

```javascript
const asignacion = 2; // 2
const suma = 2 + 3; // 5
const resta = 2 - 1; // 1
const multiplicar = 2 * 2; // 4
const dividir = 4 / 2; // 2
const negarValor = !true; // false
const residuo = 7 % 3; // 1

let init = 0;

init++; // 1
init += 1; // 2 // init = init + 1;
init--; // 1
init -= 1; // 0 // init = init - 1

init = 2;
init *= 3; // 6 // init = init * 3;
init /= 2; // 3 // init = init / 2;
```

## Comparación de valores

Mediante las siguientes operaciones podemos comparar valores de variables, siendo la respuesta siempre un booleano.

```
- valor1 == valor2
- valor1 === valor2
- valor1 > valor2
- valor1 < valor2
- valor1 >= valor2
- valor1 <= valor2
```
