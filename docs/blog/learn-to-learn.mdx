El método Feynman es un `modelo mental` que se utiliza como `técnica de comprensión` para `ayudarnos a aprender algo`, por muy complicado de entender que sea.

`Descomponiendo los conceptos en sus elementos más básicos`, para luego volver a unirlos construyendo una base sólida para la información nueva.

Esta técnica nos ayudara a ser capaces de `enseñar un concepto por complejo que sea`, a otro estudiante y otra persona que no conozca el tema.

> Tomar una información compleja y resumirla con un mensaje simple que se entienda fácilmente.

De esta forma, desachamos todas las suposiciones que teníamos en relación al conocimiento que teníamos de ese concepto que queremos aprender.

Es por eso que está `técnica de comprensión` estaremos construyendo una base sólida, lo que nos permitirá dominar conceptos complejos, con mayor confianza y eficacia.

Para utilizar esta técnica debemos realizar los siguientes pasos:

1.  Seleccionar el concepto a estudiar
2.  Desarrollar el concepto
3.  Revisar y completar
4.  Reescribir, simplificar y explicar de forma sencilla
