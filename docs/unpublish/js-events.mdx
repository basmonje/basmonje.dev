Javascript es un lenguaje que se ejecuta en el orden de lectura del archivo y una vez finalizada la carga de este, termina la ejecución.

Podemos escuchar eventos que se realicen posterior a la ejecución de nuestro código y mediante callbacks, ejecutar métodos desde memoria cuando el evento es desencadenado.

```javascript
function onBtnClick() {
    console.log('Button Clicked!');
}

document.getElementById('btn').addEventListener('click', onBtnClick);
```

## Algunos eventos

- **blur**: deseleccionar el elemento
- **change**: modificar el valor del elemento
- **input**: añadir valor al elemento
- **click**: hacer click sobre el elemento
- **dblclick**: hacer doble click sobre el elemento
- **focus**: seleccionar el elemento
- **mouseover**: poner el cursor sobre el elemento
- **mouseleave**: sacar el cursor del elemento
- **keypress**: presionar una tecla
