# Biblioteca del Sistema Operativo

# Obtener el directorio de trabajo actual

Existen dos tipos de rutas, se les llaman `rutas relativas` y `rutas absolutas`.

Una ruta absoluta se refiere a la misma ubicación en un sistema de archivos en relación con el `directorio de raíz "/"`.

Una ruta relativa se refiere a una ubicación específica en un sistema de archivos en relación con el directorio de `trabajo actual`.

```python
import os

# mostrar el directorio de trabajo actual
path_absolute = os.getcwd()

print(path_absolute)
```

# Mostrar el contenido del directorio

Para ver los archivos dentro de nuestra carpeta de proyecto.

```python
import os

_content = os.listdir()
print(_content)
```

Incluso podemos ver el contenido dentro de una carpeta específica.

```python
array_files = os.listdir('your_folder_with_path_absolute')
print(array_files)
```

# Uniendo rutas

Concatenar en python es muy simple, sin embargo, cuando queremos trabajar con distintos sistemas operativos como windows, linus o mac, tenemos algunos problemas. Esto sucede porque Mac OS y linux utilizan la barra diagonal `/` mientras que windows utiliza la barra invertida `\`. Por suerte la libreria `os` nos ayudar a facilitar este proceso, con el metodo `os.path.join()`.

```python
import os

rute_join = os.path.join(os.getcwd(), 'your_name_folder')
print(rute_join)
```

# Crear nuevo directorio

Crear carpetas no tiene mucha ciencia, solo debemos agregar:

```python
import os

os.mkdir('Dataset')
```

# Renombrar el nombre de un archivo

De manera similar a la creación de directorios, cambiar el nombre de los archivos es una tarea que podemos automatizar con un script de python.

```python
import os

os.rename("Dataset", "data")
```

Incluso podemos cambiar el nombre de varios archivos dentro de un bucle.

```python
import os

for i, file in os.listdir():
    if file.endswith('.csv'):
        os.rename(file, f'2021_{file}')
```

# Verificar archivo existente

Puede revisar si un archivo o directorio existe o no con el módulo `os`.

```python
import os

q = os.path.exists('data')
print(q)
```

Como también podemos agregarlo en una condición.

```python
import os

if os.path.exists('data'):
    print('Si existe la carpeta o archivo')
else:
    print('Lo siento, no se encuentra lo que buscar')
```

# How to use Conda in Windows 10

Crear una nuevo entorno de trabajo.
Create a new environ of work.

```
conda create -n myenv python=3.6
```

Para activar entorno creado.

```
conda activate myenv
```

Para conseguir una lista de nuestros entornos creados.

```
conda env list
```
