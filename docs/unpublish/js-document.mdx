Es el API de javascript que nos permite interactuar con el DOM.
Las etiquetas HTML se añaden como propiedades al objeto
document.

```javascript
console.log(document);
```

Algunos usos comunes en que se utiliza document es para obtener elementos específicos de nuestro DOM y manipularlos, o obtener información referente a estos elementos.

```javascript
// seleccionar un elemento
document.getElementsByTagName("p");
document.getElementById("#some-id");
document.getElementsByClassName(".some-class");
document.querySelector("some-selector");
document.querySelectorAll("some-selector");

// anadir estilo inline a un elemento
const body = document.getElementsByTagName("body");
body[0].style = "background-color: #c4c544;";

// interactuar con las clases del elemento
body[0].classList.add("some-class");
body[0].classList.remove("some-class");
body[0].classList.toggle("some-class");

// obtener el valor de un input
const input = document.getElementsByTagName("input");
console.log(input[0].value);
```
