Las siguientes configuraciones son para el archivo `next.config.js`.

## Agregar SCSS

```js:next.config.js
const path = require("path");

module.exports = {
  sassOptions: {
    includePaths: [path.join(__dirname, "styles")],
  },
};
```

## Agregrar alias de Webpack con Nextjs

```js:next.config.js
const path = require("path");

module.exports = {
  webpack: (config) => {
    config.resolve.alias._components = path.join(__dirname, "src/components");
    config.watch = false;
    return config;
  },
};
```
> Existe otra forma de crear un alias con vscode configurando el archivo [jsconfig.json](/snippets/config-jsconfig#agregar-alias)
