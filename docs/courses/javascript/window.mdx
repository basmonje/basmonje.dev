Representa la ventana que contiene al DOM y nos permite realizar acciones sobre la misma ventana del 
navegador.

```javascript
window.open('http://google.com');
// abre una nueva ventana

window.close();
// cierra la ventana actual

window.confirm();
// muestra emergente de confirmacion con 2 opciones

window.alert();
// muetra emergente de alerta con 1 opcion
```

## Variables en window

window puede alojar variables o funciones que quedan disponibles de forma global como una propiedad del objeto. Es posible acceder a las propiedades de window sin necesidad de especificar el objeto.

```javascript
window.myCustomVar = 'hola';

console.log(window.myCustomVar); // 'hola'
console.log(myCustomVar); // hola
```
